     # Verba

     ## Setup
     1. Create a virtual environment:
        python -m venv verba-env
     2. Activate it:
        verba-env\Scripts\activate  # (Windows)
     3. Install dependencies:
        pip install -r requirements.txt
     4. Run the backend:
        python backend/api.py
     5. Serve the frontend (optional):
        python serve_frontend.py

     ## Notes
     - Do not upload your virtual environment or database files.
